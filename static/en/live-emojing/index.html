I would like to make a proposal to you:
A show of music with emojis! 

How is that?


It is an  audiovisual performance that is live coded . The code is projected so that it is seen how it is manipulated to generate music and visuals.💻🎼

 People can participate  (even if they do not code)   by sending emojis from their phone 📱😎😱🤖. These emojis get to the code 📟👾and modify the music we code alltogether 🎹🎷🎉😂🎊

 The arrengements are simple . I take my notebook with me💻 and only need projector, sound and internet. 📺🔊📶
